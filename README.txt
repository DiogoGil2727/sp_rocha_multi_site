SP ROCHA — site multi-páginas (pastas)

Como correr:
- Abre index.html no browser (duplo clique).
- Para meter online: faz upload desta pasta para Netlify / Vercel (static) / GitHub Pages / qualquer hosting.

Como editar rápido:
- HOME: index.html (cards das pastas)
- Páginas: roda-foz.html, roda-centro.html, musicas.html
- CSS: assets/css/styles.css
- Imagens: assets/img/

YouTube:
- Troca VIDEO_ID_AQUI pelo ID do YouTube (a parte depois de v=).
Exemplo:
https://www.youtube.com/watch?v=dQw4w9WgXcQ
ID = dQw4w9WgXcQ
Embed:
https://www.youtube.com/embed/dQw4w9WgXcQ
